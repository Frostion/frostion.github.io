website history

while pushing an update to my website recently, i realized that because my site is hosted on a github repository, its entire history has been stored in the repo's commit history. i've wildly redesigned my website many, many times since i bought my first domain, so i thought it would be fun to revisit all those previous designs.

nov 2, 2016 - very first website

my first website ever! i had no idea how to write HTML or CSS, so i just embraced the 90s my-first-website look fully. the only links that worked were "about me" and "news".

feb 2, 2017

i started learning some basic CSS and layout principles, so i made something that looked a bit more "modern" (bland and boring). i developed a pokemon plushie hyperfixation around this time, so this site's main purpose was to host reviews and photos of my plushie collection. i also started getting interested in building my own synthesizers at this time.

jun 7, 2017

this major redesign occurred because i actually learned some CSS animation basics! i was super proud of those hover animations on the index page. to be fair to past frost, they are pretty cool. i also became a blue, icy, black-backed jackal at this time. (still a long way off from discovering my true dragon self, but at least i realized i wasn't human by now.)

mar 30, 2018

this entire page is just one long scrolling thing, which is actually sorta cool! it's missing a detailed "about me" section, which is somewhat of a downgrade.

dec 10, 2018

at this time i thought that having a single, ever-lengthening, scrolling page would become too awkward if i added too much stuff to it, so i once again divided the sections up into their own separate pages. the detailed "about me" section hasn't returned yet, though.

i was considering taking custom commissions for digital animated fursona badges at this point because so many people told me that They Would Totally Buy One, but i since decided against it because selling people hand-made electronic devices is not something i want to do.

apr 3, 2021

the actual layout of this site is very similar to my previous one, but the aesthetics have been MASSIVELY upgraded. check out those sick hexagons and rad fonts! i'm finally starting to find my style at this point.

feb 28, 2022

this iteration of my site occurred when i first heard about the "web revival" movement. it's by far the most "real" a website of mine has been up to this point.

jul 18, 2022 - current design

the curent design of my website. it's definitely my favorite layout to date.
