widget xml generator

this tool can help you generate the required myloConfig.xml and widgetPackage.xml files for a widget.

widget info

the name of the widget displayed in the widget installer.

a short description of the widget.

the date that you created this version of the widget.

the current revision of the widget.

the mylo must be running at least this OS version in order to use your widget. for maximum compatibility, keep this at 1.000 whenever possible.

two-letter locale code for the language of the widget.

the mylo will try to download the widget from this URL whenever it checks for updates. due to the TLS Apocalypse, this must be http, not https.

author info

the name of the widget's author displayed in the widget manager.

the website of the widget's author displayed in the widget manager. ideally, this would point to the page on your website that mentions the widget (if you have one).

widget size

* these units are not pixels - 64 units is equal to the width of the mylo's screen.
