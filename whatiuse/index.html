software

art and image manipulation

  • affinity designer • vector graphics editor that i use for all of my artwork.
  • affinity photo • photo manipulation program that i use for developing raw photos.
  • bryce 7 • 3D modeling software and ray-tracing renderer tailored towards creating otherworldly, procedurally-generated environments. most of the 3D backgrounds in my artwork are created with this program.
  • paint.net • free image editing program that's responsive and remarkably easy to use, while still having an impressive featureset and expansive plugin library.

computer customization

  • stardock windowblinds • allows for applying custom themes to windows that can customize every part of the system UI. i primarily use the alter ego skin on my computers.
  • stardock start10 • brings back classic start menus to windows 10, and works in tandem with windowblinds to allow the start menu to be themed too.
  • microsoft powertoys • a suite of useful utilities that extend the functionality of windows. i highly recommend checking out powertoys run in particular - it's basically spotlight search but for windows.
