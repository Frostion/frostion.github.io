software.

web apps.

Utilities and toys that run right in your web browser!

LCD Simulator

Utility for creating screenshots and mockups of HD44780-based character LCDs. Features custom character support, arbitrary LCD sizes, and a handful of color palettes to choose from.

Cyberdragon Weather

Daily and hourly weather forecast for your current location. Sources data straight from the National Weather Service for the most accurate forecast. (Only works in the USA, sorry!)

Ikea Shopping List

The shopping list function on Ikea's website is annoyingly feature-free, so I wrote my own (much better) one that actually shows the location and stock of each item.

Taco Bell Item Generator

Ever feel like the items on Taco Bell's menu are randomly generated? I sure do, so I wrote my own Taco Bell item generator to prove a point or something.

windows apps.

Software and utilities for Windows. These are written in C#.NET and are all open source on GitHub.

Audio Snipping Tool

Records any audio that your computer is playing for quick saving, sharing, and drag-and-dropping. It's basically the Windows snipping tool, but for audio.

VRQR

QR code scanner for VRChat. Just have this program running while you're in VRChat, and it will automatically scan all photos you take with the in-game camera for QR codes.

Unity Package Extractor

Simple utility that lets you explore and yoink files out of a Unity package without having to go through the painful process of actually launching Unity.

cybiko apps.

Programs and games for the Cybiko wireless inter-tainment system. These are written in B2C and are all open source on GitHub.
(Hey, if you're interested in the Cybiko, you should join my Cybiko discord server!)

Subiko

Fully-featured sudoku game for the Cybiko with auto candidate calculation and removal, incorrect move identification, and autofill. It even comes preloaded with 45 puzzles from the NY Times.
