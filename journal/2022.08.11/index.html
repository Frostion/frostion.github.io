i've finally customized my phone's launcher in a way that i'm happy with! i have a unihertz titan pocket, a fantastic phone that i really should write a review about sometime. even though it's a smartphone, i just use it like a feature phone with the added ability to connect to IM services (like discord and telegram), so today i decided to flex my customization muscles and create a home screen for it that gives off that feature phone vibe.

to achieve this, i used smart launcher, simple text widget for the folder labels, and the rondo icon pack for a unified look.

here's the home screen. i'm pretty happy with it!
