i've finally customized my phone's launcher in a way that i'm happy with! i have a unihertz titan pocket, a fantastic phone that i really should write a review about sometime. even though it's a smartphone, i mostly just use it as a feature phone with IM functionality, so today i decided to create a custom home screen for it that gives off that 2010s feature phone vibe. i think it turned out pretty well!

to achieve this, i used smart launcher, simple text widget for the folder labels, and the rondo icon pack for a unified look.
