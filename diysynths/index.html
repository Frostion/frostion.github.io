Arduino/Electronic Synths

Sequin

Chiptune Melody Synthesizer

(Some more info will magically appear here at some point in the future.)

A tiny Arduino synth that works great as a signal source for an effect pedal chain. It's a 4 step sequencer that creates Nintendo/Commodore 64-style sounds and melodies. Features pulse width modulation, a built-in bitcrusher, and arpeggiation.
YouTube demos should be coming soon.

Axoloti Patches

All of these programs run on the Axoloti programmable synth.

Chipwave

Chiptune Synth and Drum Sequencer

.ZIP download

This patch contains many features for creating chiptune music live. It features a simple drum synth and sequencer, an NES-like triangle-wave monophonic bass section, a PWM square-wave lead section with 6 voices, and a fast arpeggio section with 6 voices. In the .ZIP file, there is a "Euclidean in" project that needs to be in your "axoloti/objects/" folder before running the Chipwave patch.
