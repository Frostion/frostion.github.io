link directory

other cool places on the web that i think are worth checking out!

websites and blogs

  • GUIdebook • an expansive gallery of various graphical user interfaces and their icons, splash screens, sounds, etc.
  • old vintage computing research • a blog containing lots of interesting vintage computing projects, stories, and research, usually about lesser-known and more interesting platforms.

communities

  • melonland forum • a web forum loosely based around making virtual worlds, websites, and other creative projects. it's a super chill place to hang out and chat!
  • status cafe • a website where you can post short little status updates and browse through other members's updates. the status widget in the right sidebar of my website is powered by this!

resources and tools

  • dumbphone finder • a database of modern feature phones ("dumbphones") that can be filtered by OS, form factor, features, and network compatibility. very helpful for finding your next phone!
  • getting updates and software for old windows versions • a massive list of the last windows xp-compatible versions of various software, complete with download links.

youtube

  • building an 8-bit breadboard computer • a massive 44-part series by Ben Eater that teaches every little detail of how a microprocessor works by demonstrating how to build one from scratch using basic logic chips on a breadboard.
  • Kassio Project - ETERNAL RHYTHM • a glimpse into an alternate, hyper-virtual universe, in music video form.
  • thorhighheels • a video game review/showcase channel that covers older, very obscure games from a more artistic perspective.
